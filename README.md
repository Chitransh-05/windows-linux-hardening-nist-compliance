# 🛡️ Endpoint Security Hardening & NIST Compliance

## 📌 Description

Security hardening and NIST SP 800-53 compliance assessment of Windows and macOS endpoints, including remediation and enterprise policy design (BYOD, email, cloud).

---

## 📖 Overview

This project focuses on evaluating and securing enterprise endpoints in a hybrid/cloud environment. It involves identifying misconfigurations in Windows and macOS systems, mapping them against **NIST SP 800-53 Rev. 5 controls**, and implementing remediation strategies to enhance overall security posture.

Additionally, the project includes the development of **enterprise security policies** and **secure cloud server configurations**.

---

## 🎯 Objectives

* Perform endpoint **security hardening (Windows & macOS)**
* Assess systems against **NIST SP 800-53 controls**
* Identify and remediate security gaps
* Develop **enterprise-grade security policies**
* Improve compliance and reduce attack surface

---

## 🧪 Project Components

### 🔹 Windows Security Hardening

* Evaluated configurations like UAC, Firewall, BitLocker, Audit Policies
* Checked compliance with NIST controls
* Implemented remediation steps

### 🔹 macOS Security Configuration

* Applied security controls including:

  * FileVault encryption
  * Gatekeeper enforcement
  * Firewall & logging
  * Strong authentication policies

### 🔹 NIST Compliance Assessment

* Evaluated 14 controls from NIST SP 800-53 Rev. 5
* Classified as: **Met / Not Met / Not Applicable**
* Provided actionable remediation steps

### 🔹 Security Policy Design

* 📧 Email Security Policy
* 📱 BYOD Policy
* ☁️ Cloud Server Build Standards
* 🔐 CASB Security Enhancements
